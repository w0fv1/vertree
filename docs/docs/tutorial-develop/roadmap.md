# 后续规划

0.8.0 之后，Vertree 的重点不再是“把基础功能做出来”，而是把现有能力打磨得更稳定、更容易分发。

## 1. Linux 支持

Windows 和 macOS 已经具备桌面主流程，下一步重点会转向 Linux：

- 基础桌面运行
- 托盘与通知适配
- 常见发行版的打包方式

## 2. 版本树交互与性能

- 优化复杂树结构下的布局稳定性
- 继续改进画布拖拽、缩放和连接线绘制
- 降低大树场景下的重绘压力

## 3. 平台集成细化

- Windows 11 新菜单分发体验继续优化
- macOS 发布包、签名与分发流程补齐
- 各平台权限请求尽量做到按需触发

## 4. 文件能力增强

- 版本搜索
- 标签/备注管理增强
- 差异展示能力探索

## 5. 文档与发布流程

- 补齐升级说明
- 统一 README、站点文档和 Release Notes
- 让发布产物、版本号和文档内容始终同步
